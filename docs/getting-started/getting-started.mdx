# Getting Started

The [widgetbook package](https://pub.dev/packages/widgetbook) provides a Flutter Widget called `Widgetbook` in which custom Widgets from your app can be injected.

## Approaches

Widgetbook can be configured using two different approaches. 

The **Manual** approach uses the [widgetbook package](https://pub.dev/packages/widgetbook) and especially requires manual maintanance to keep the Widget structure in Widgetbook in sync with your application. 

The **Generator** approach uses the [widgetbook_generator package](https://pub.dev/packages/widgetbook_generator) packages in combination with the [widgetbook_annotation package](https://pub.dev/packages/widgetbook_annotation) to automatically configure your Widget structure in Widgetbook to the structure defined in your application. Therefore, this approach keeps your Widgetbook in sync with your app.

<Info>The generator approach is recommended over the Manual approach since it simplified maintanance of Widgetbook. In addition, the Generator approach has integration with [Widgetbook Cloud](/widgetbook-cloud/overview).</Info>

## Install dependencies

Make sure to depend on the Widgetbook packages.

  values={[
    { label: 'Generator', value: 'generator' },
    { label: 'Manual', value: 'manual' },

    ```bash
    flutter pub add widgetbook_annotation
    flutter pub add widgetbook widgetbook_generator build_runner --dev
    ```

## Setup an empty Widgetbook

<Tabs
  groupId="approach"
  values={[
    { label: 'Generator', value: 'generator' },
    { label: 'Manual', value: 'manual' },
  ]}
>
  <TabItem value="generator">

    Create a new file in your `lib` folder called `app.dart` or annotate an existing code element using the `@WidgetbookApp` annotation. 

    <Info>Dependending of the file in which `@WidgetbookApp` is used, a corresponding `.widgetbook.dart` file will be created by the [widgetbook_generator package](https://pub.dev/packages/widgetbook_generator). The annotated code element does not have an inpact of the generated output code.</Info>

    ```dart title="Setup of Widgetbook: app.dart"
    @WidgetbookApp.material()
    late int notUsed;
    ```

    Run the [build_runner](https://pub.dev/packages/build_runner) by executing `flutter run build_runner build`.
    The command generates the `app.widgetbook.dart` file. 

    ```dart title="Generated Widgetbook file: app.widgetbook.dart"
    import 'package:flutter/material.dart';
    import 'package:widgetbook/widgetbook.dart';

    void main() {
      runApp(HotReload());
    }

    class HotReload extends StatelessWidget {
      const HotReload({Key? key}) : super(key: key);

      @override
      Widget build(BuildContext context) {
        return Widgetbook.material(
          addons: [],
          directories: [],
        );
      }
    }
    ```

  </TabItem>
  <TabItem value="manual">
    Create a new file `app.widgetbook.dart` in your `lib` folder.

    ```dart title="Widgetbook file: app.widgetbook.dart"
    import 'package:flutter/material.dart';
    import 'package:widgetbook/widgetbook.dart';

    void main() {
      runApp(const HotReload());
    }

    class HotReload extends StatelessWidget {
      const HotReload({Key? key}) : super(key: key);

      @override
      Widget build(BuildContext context) {
        return Widgetbook.material(
          addons: [],
          directories: [],
        );
      }
    }
    ```
  </TabItem>
</Tabs>

## Run the empty Widgetbook

Check if everything works by running `flutter run` with the newly created Widgetbook:
