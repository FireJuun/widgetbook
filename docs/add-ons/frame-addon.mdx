# Overview

The frame Add-on provides `Frame`s for previewing your use-cases.
A frame can have different functionality but often comes with `Device`s that give width and hight constraints to the previewed component by overriding `MediaQuery` parameters.

## Available frames

| Frame              | Has Devices | Description |
| ------------------ | ----------- | ----------- |
| DefaultDeviceFrame | ✅ | A frame based on the [device_frame package](https://pub.dev/packages/device_frame) |
| WidgetbookFrame    | ✅ | A minimal, borderless frame |
| NoFrame            | ➖ | A device-less frame that resized whenever the Widgetbook window is changed |

## Specify

Specify the `Frame`s and (if available) the `Frame`'s `Device`s for the frame Add-on.

<Tabs
  groupId="approach"
  values={[
    { label: 'Generator', value: 'generator' },
    { label: 'Manual', value: 'manual' },
  ]}
>
  <TabItem value="generator">
    <Warning>
      As of now, the generator always sets up all available `Frame`s. 
    </Warning>

    `Frames` are defined over `Devices` and are specified via the `@WidgetbookApp` annotation.
    ```dart title="Example: Frames with devices"
    @WidgetbookApp.material(
      devices: [
        Apple.iPhone12,
        Apple.iPhone13,
        Apple.iPadMini,
        Desktop.desktop1080p,
      ],
    )
    class App extends StatelessWidget { ... }
    ```
  </TabItem>
  <TabItem value="manual">
    First, make sure to define the devices you want to use.
    ```dart title="Example: Devices"
    final devices = [
      Apple.iPhone12,
      Apple.iPhone13,
      Apple.iPadMini,
      Desktop.desktop1080p,
    ];
    ```

    Second, define the frame Add-on.
    ```dart title="Exmaple: Setup frame Add-on"
    final frameAddon = FrameAddon(
      setting: FrameSetting(
        frames: [
          NoFrame(),
          DefaultDeviceFrame(
            setting: DeviceSetting(
               devices: devices,
               activeDevice:  Apple.iPhone13,
            ),
          ),
          WidgetbookFrame(
            setting: DeviceSetting(
               devices: devices,
              activeDevice: Apple.iPhone13,
            ),
          ),          
        ],
        activeFrame: NoFrame(),
      ),
    ),
    ```
  </TabItem>
</Tabs>

## Custom devices

You can define your own custom device by using the `Device` class:

```dart
Device(
  name: 'Custom Device',
  resolution: Resolution.dimensions(
    width: 500,
    height: 500,
    scaleFactor: 2,
  ),
  type: DeviceType.tablet,
),
```
