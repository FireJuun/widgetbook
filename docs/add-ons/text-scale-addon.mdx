# Overview

The text scale Add-on lets users change the active text scale for previewing your use-cases.
Previwing different text scales can be interesting for teams as the layout changes due to text scale and therefore text size.

## Setup

<Tabs
  groupId="approach"
  values={[
    { label: 'Generator', value: 'generator' },
    { label: 'Manual', value: 'manual' },
  ]}
>
  <TabItem value="generator">
    As text-scales are just defined as a `double` value, the available text scales within Widgetbook are defined via the `@WidgetbookApp` annotation. 

    ```dart title="Example: Text scales"
    @WidgetbookApp.material(
      textScaleFactors: [1, 1.5, 2],
    )
    class App extends StatelessWidget { ... }
    ```
  </TabItem>
  <TabItem value="manual">
    Create your text scale Add-on using the `TextScaleAddon` and `TextScaleSetting` classes:
    ```dart title="Example: Create the text scale Addon"
    final textScaleAddon = TextScaleAddon(
      setting: TextScaleSetting(
        textScales: [
          1.0,
          1.25,
          1.5,
        ],
        activeTextScale: 1,
      ),
    ),
    ```
  </TabItem>
</Tabs>
